Was able to connect to the nalu scientific board using the NaluScope software (manual, download).

Here are the steps I did on Windows 10:

  1. Make Hardware connections
    1. Connect Nalu's HDSoCv1 evalr2 board to the FMC of a Nexys Video A7 board (you can follow the quickstart guide for this step)
    2. Connect a USB to microUSB cable to the Nexys Video A7 board PROG port.
    3. Connect a USB to microUSB cable to the Nexys Video A7 board UART port.
    4. Connect the 12V power adapter to the board's power input.
    5. Switch the board on. You should see lights on both boards (the Nexsys and HDSoC).
  2. Open Vivado to upload bitstream
    1. Go to hardware manager
    2. Autoconnect to the device, you should see something like xc7a200t_0. This is the Nexys board.
    3. Click program device. Select the bitstream (Firmware) downloaded from the manual page, or this link will download it too.
    4. You should see some lights change on both boards, you can see what they mean in the quickstart guide
  3. Open the NaluScope software for use
    1. Choose a working directory
    2. Select model hdsocv1_evalr2
    3. Leave "Use Hardware Server" and "Connect to Board" checked
    4. Under "Connect to Board", select FTDI
      1. There should be 2 devices listed, one for the programming USB and one for the UART usb. Pick the one that has UART in it's name (mine was named AV0KR3DC or something)
    5. Click "Connect" wait ~1 minute
    6. Once you see "Startup successful" click "Continue" at the bottom. A new window should appear

Here are the steps I followed to reproduce the behavior we had on windows:

  1. Make Hardware connections

    1. Connect Nalu's HDSoCv1 evalr2 board to the FMC of a Nexys Video A7 board (you can follow the quickstart guide for this step)
    2. Connect a USB to microUSB cable to the Nexys Video A7 board PROG port.
    3. Connect a USB to microUSB cable to the Nexys Video A7 board UART port.
    4. Connect the 12V power adapter to the board's power input.
    5. Switch the board on. You should see lights on both boards (the Nexsys and HDSoC).

  2. Open Vivado to upload bitstream

    1. Go to hardware manager
    2. Autoconnect to the device, you should see something like xc7a200t_0. This is the Nexys board.
    3. Click program device. Select the bitstream (Firmware) downloaded from the manual page, or this link will download it too.
    4. You should see some lights change on both boards, you can see what they mean in the quickstart guide

  3. Install NaluScope for Linux

    1. Download latest Linux version off nalu's website
    2. Extract naluscope tar -xzvf naluscope*.tar.gz
    3. Download dependencies:
    sudo apt update
sudo apt install libxcb-cursor0 libxcb-icccm4 libxcb-keysyms1 libxcb-render-util0 libxkbcommon-x11-0
    sudo apt update
sudo apt install libxcb-cursor0 libxcb-icccm4 libxcb-keysyms1 libxcb-render-util0 libxkbcommon-x11-0

    1. Download USB to UART Driver (D2XX Driver)

    2. The NaluScope Manual suggests installing under home as root, so just mv nalu/ $HOME.

  4. Open the NaluScope software for use

    1. ./$HOME/nalu/nalu (It may complain about not having D3XX Driver, this driver is for USB3 and isn't supported on linux anyways)
